Belajar Web Development

Web development adalah proses membangun dan mengelola website. Ada tiga elemen utama dalam web development:

  1. HTML: Struktur dasar dari halaman web.
  2. CSS: Digunakan untuk memberikan gaya (style) pada halaman web.
  3. JavaScript: Memberikan interaktivitas pada website.

Tools yang Dibutuhkan

  • Code Editor: Visual Studio Code, Sublime Text
  • Browser: Google Chrome, Mozilla Firefox

Contoh Kode HTML Sederhana

<!DOCTYPE html>
<html>
<head>
    <title>Hello World</title>
</head>
<body>
    <h1>Halo Dunia!</h1>
</body>
</html>
